◆ find_pose()

vision-node.find_pose ( point_cloud,
block ) 
@brief Function to compute object pose. Basically, the block is "sliced" at a specific height to find the 2
useful points:
- x_min: the nearest point the camera
- y_min: the rightmost point (near the arm)
- y_max: the leftmost point (furthest from the arm)
these points are used to calculate yaw angle
@param point cloud and the block
@retval yaw angle (radiant)
